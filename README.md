# passwordmanager
A Password Manager I have built for myself using Python and JavaScript.

Prerequisites:
    - Python 3.6 or Newer
    - Flask==0.12.2
    - Flask-JWT-Extended==3.12.1
    - passlib==1.7.1

Installing:
    - git clone https://github.com/wproffi2/passwordmanager
    - For Windows:
        - Run setup.bat
        - Run launch.bat
