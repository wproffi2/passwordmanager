# passwordmanager
A Password Manager I have built for myself using Python and JavaScript.

# Heading 1
Prerequisites:
* Python 3.6 or Newer
* Latest version of PIP 
* Flask==0.12.2
* Flask-JWT-Extended==3.12.1
* passlib==1.7.1

Installing:
1. git clone https://github.com/wproffi2/passwordmanager
2. 
    * For Windows:
        1. Run setup.bat
        2. Run launch.bat
